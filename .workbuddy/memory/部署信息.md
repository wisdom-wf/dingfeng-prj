# 部署信息

## 当前状态

- **本地运行端口**：11236
- **访问地址**：http://127.0.0.1:11236
- **部署方式**：直接运行 Python Flask

## 启动命令

```bash
cd /Volumes/works/my-projects/10-dingfeng--pro/web_app
python3 app_modern.py
```

## systemd 服务配置（可选）

```ini
[Unit]
Description=Dingfeng Reservation Data Collector
After=network.target

[Service]
Type=simple
User=ubuntu
WorkingDirectory=/path/to/web_app
ExecStart=/path/to/venv/bin/python /path/to/web_app/app_modern.py
Restart=always

[Install]
WantedBy=multi-user.target
```

## Nginx 反向代理配置（可选）

如果需要通过域名访问：

```nginx
location /dingfeng {
    proxy_pass http://127.0.0.1:11236;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
}
```

## 注意事项

- 数据仅存储在内存中，重启后丢失
- 如需持久化可考虑添加文件或数据库存储
