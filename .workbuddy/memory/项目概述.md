# 丁峰家政管理系统项目概述

## 基本信息

- **项目名称**：丁峰家政管理系统（dingfeng-prj）
- **仓库**：`https://github.com/wisdom-wf/dingfeng-prj.git`
- **项目路径**：`/Volumes/works/my-projects/10-dingfeng--pro`
- **当前版本**：v2.0

## 技术栈

- **后端**：Python 3 + Flask
- **前端**：Bootstrap 5 + Font Awesome + ECharts
- **数据处理**：Pandas
- **API**：RESTful API
- **端口**：11236

## 主要功能

养老服务预约数据采集与分析系统：
1. 数据采集 - curl 命令从全国养老服务信息平台 API 采集
2. 自动刷新监控 - 每 120 秒自动刷新
3. 多数据源同时监控
4. 数据可视化（饼图、柱状图、词云、地区分布）
5. CSV/Excel 导出
6. 身份证/手机号脱敏
