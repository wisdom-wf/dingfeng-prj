# 养老服务预约数据采集与分析系统 - 使用说明

## 🎯 系统概述

这是一个现代化的养老服务预约数据采集与分析系统，可以从全国养老服务信息平台自动采集数据并进行智能分析和可视化展示。

## 🚀 系统特点

### 🔧 核心功能
- **数据采集**: 通过curl命令从官方API采集预约数据
- **智能分析**: 自动生成多维度统计分析
- **可视化展示**: 丰富的图表和数据表格展示
- **多种导出**: 支持CSV、Excel格式数据导出
- **实时监控**: 进度条和状态反馈

### 🎨 界面特色
- 现代化的响应式设计
- 直观的操作流程
- 实时数据可视化
- 优雅的加载动画

## 📋 使用步骤

### 第一步：获取curl命令
1. 在浏览器中登录全国养老服务信息平台
2. 按 `F12` 打开开发者工具
3. 切换到 **Network** 面板
4. 刷新页面，找到 `reservationPage` 请求
5. 右键点击请求 → **Copy** → **Copy as cURL (bash)**

### 第二步：粘贴并采集
1. 在系统界面中粘贴复制的curl命令
2. 点击"开始采集数据"按钮
3. 等待系统自动完成数据采集和分析

### 第三步：查看和导出
1. 系统会自动显示统计卡片和图表
2. 可以查看详细的数据表格
3. 点击"导出CSV"或"导出Excel"保存数据

## 🔧 API接口

### 数据采集接口
```
POST /api/collect
Content-Type: application/json

{
    "curl": "curl 'https://ylfw.mca.gov.cn/...' -H ..."
}
```

### 数据导出接口
```
POST /api/export/csv     # 导出CSV
POST /api/export/excel   # 导出Excel
```

### 状态查询接口
```
GET /api/stats           # 获取统计数据
GET /api/recent-data     # 获取最近数据
GET /health              # 健康检查
```

## 📊 数据分析内容

系统会自动分析以下维度：

### 基础统计
- 总预约数量
- 未处理/已处理数量
- 服务类型分布
- 时间趋势分析

### 高级分析
- 地址分布热力图
- 区域数据统计
- 服务类型占比
- 处理时效分析

## ⚠️ 注意事项

### 安全提醒
- curl命令包含认证信息，请妥善保管
- 建议定期更新认证token
- 生产环境请使用HTTPS

### 性能优化
- 大量数据采集时请耐心等待
- 建议分批次采集历史数据
- 导出大文件时可能需要较长时间

### 技术要求
- Python 3.8+
- 现代浏览器（Chrome/Firefox/Safari最新版）
- 稳定的网络连接

## 🆘 常见问题

### Q: curl命令过期怎么办？
A: 重新登录平台，按照第一步重新获取新的curl命令

### Q: 采集速度很慢？
A: 这是正常的，系统为了保护服务器采用了合理的请求间隔

### Q: 数据导出失败？
A: 检查是否有足够的磁盘空间，或尝试导出较小的数据集

### Q: 图表不显示？
A: 确保网络连接正常，某些图表需要外部API支持

## 📞 技术支持

如遇到问题，请检查：
1. Python环境和依赖包是否正确安装
2. 网络连接是否稳定
3. curl命令是否正确且未过期
4. 浏览器控制台是否有错误信息
