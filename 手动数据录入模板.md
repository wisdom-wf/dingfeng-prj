# 手动数据录入方案

如果自动化抓取困难，建议采用手动录入 + 自动分析的方案。

---

## 📋 方案一：Excel手动录入（推荐）

### 步骤1：创建Excel文件

创建一个Excel文件，包含以下列：

| 姓名 | 公民身份号码 | 联系电话 | 预约服务类型 | 预约时间 | 处理状态 |
|------|-------------|----------|-------------|----------|----------|
| 梁秀英 | 610621194206302220 | 15991414829 | 上门服务 | 2026-03-19 上午 | 未处理 |
| 冯彦树 | 612601194512260918 | 15191153498 | 上门服务 | 2026-03-19 上午 | 未处理 |
| ... | ... | ... | ... | ... | ... |

### 步骤2：运行分析脚本

```bash
python3 Excel数据分析.py 你的文件.xlsx
```

---

## 📋 方案二：网页复制粘贴

### 步骤1：复制数据

1. 在预约信息管理页面，选中表格数据
2. 按 `Ctrl+C` (Windows) 或 `Cmd+C` (Mac) 复制
3. 粘贴到文本文件或直接运行整理工具

### 步骤2：运行整理工具

```bash
python3 数据整理工具.py
```

按提示粘贴数据，工具自动解析并生成报告。

---

## 📋 方案三：截图OCR识别

如果复制不方便，可以：

1. 截图保存表格页面
2. 使用OCR工具（如微信、QQ的截图文字识别）提取文字
3. 整理成Excel或直接粘贴到整理工具

## 💡 建议

考虑到数据量不大（截图显示11条），**手动录入Excel可能是最快的方式**：

1. 打开Excel
2. 创建6列：姓名、公民身份号码、联系电话、预约服务类型、预约时间、处理状态
3. 分别录入【未处理】和【已处理】的数据
4. 保存为 `.xlsx` 文件
5. 运行分析脚本生成报告

整个过程大约需要 **10-15分钟**。
