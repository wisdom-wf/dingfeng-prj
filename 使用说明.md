## 🚀 推荐方案：Excel手动录入 + 自动分析

由于浏览器自动化在这个网站上不稳定，**推荐以下简单可靠的方案**：

### 方案：Excel录入（最快10分钟完成）

#### 步骤1：创建Excel文件

打开Excel，创建6列：

| 姓名 | 公民身份号码 | 联系电话 | 预约服务类型 | 预约时间 | 处理状态 |
|------|-------------|----------|-------------|----------|----------|

#### 步骤2：录入数据

1. 在【未处理】标签页，逐行复制数据到Excel
2. 处理状态列填：**未处理**
3. 切换到【已处理】标签页，继续录入
4. 处理状态列填：**已处理**

#### 步骤3：保存Excel

保存为 `.xlsx` 格式，例如：`预约数据.xlsx`

#### 步骤4：运行分析工具

```bash
python3 Excel数据分析.py 预约数据.xlsx
```

会自动生成可视化HTML报告。

## 🔧 其他方案

### 方案二：文本复制粘贴

如果不想用Excel，可以直接复制粘贴：

```bash
python3 数据整理工具.py
```

按提示粘贴从网页复制的数据。

### 方案三：API接口抓取（技术方案）

如果能从浏览器开发者工具获取API信息：

```bash
python3 预约数据抓取_API方案.py
```

需要配置API地址、Token、Cookie等信息。

## 📊 数据分析内容

抓取的数据将包含以下字段：

| 字段 | 说明 |
|------|------|
| 姓名 | 预约人姓名 |
| 公民身份号码 | 身份证号（已脱敏处理） |
| 联系电话 | 联系电话 |
| 预约服务类型 | 服务类型（如：上门服务） |
| 预约时间 | 预约日期和时段 |
| 处理状态 | 未处理 / 已处理 |

## ⚠️ 注意事项

1. **数据安全**：所有数据保存在本地，不会上传到任何服务器
2. **敏感信息**：分析报告中身份证号已脱敏处理
3. **Excel格式**：请确保列名与模板一致

## 🔧 常见问题

### Q1: 提示缺少依赖
运行：`pip3 install pandas plotly openpyxl`

### Q2: Excel文件无法读取
确保文件是 `.xlsx` 格式，且列名正确
